# Documentation

This directory keeps user-facing guides, release notes, implementation plans, and support material for CC Desktop Switch.

## User Guides

- [Quick Start](QUICK_START.md)
- [Usage Guide](USAGE.md)

## Releases

Release notes are grouped under [`releases/`](releases/). Use `v<version>.md` for new releases, for example:

```text
docs/releases/v1.1.1.md
```

The GitHub Release workflow reads release notes from this directory.

## Plans And Reports

- [`plans/`](plans/) stores implementation plans that are still useful as code-backed design records.
- [`reports/`](reports/) stores investigation reports and issue writeups.
- [`promo/`](promo/) and [`tutorial/`](tutorial/) store public-facing screenshots and promotional material.
