# CC Desktop Switch v1.0.16

## 中文

本次版本新增阿里云百炼 Token Plan 预设，方便已购买阿里云令牌计划的用户快速配置。

### 主要变化

- **新增阿里云百炼 (Token Plan) 预设**：
  - 预设 ID：`bailian-token-plan`
  - 默认 Base URL：`https://token-plan.cn-beijing.maas.aliyuncs.com/apps/anthropic`
  - 认证方式：`bearer`（对应 `ANTHROPIC_AUTH_TOKEN`）
  - 模型映射默认留空，由用户自行配置；`default` 预填 `qwen3.6-plus` 作为参考。
  - 与现有的按量付费版「阿里云百炼」并列显示，方便区分两种计费方式。

### 下载建议

- Windows 用户可以下载 `CC-Desktop-Switch-v1.0.16-Windows-Setup.exe` 或 `CC-Desktop-Switch-v1.0.16-Windows-Portable.zip`。
- macOS 用户可以下载 `CC-Desktop-Switch-v1.0.16-macOS-arm64.pkg` 或 `CC-Desktop-Switch-v1.0.16-macOS-arm64.dmg`。

## English

This release adds the Alibaba Cloud Bailian Token Plan preset for users who have purchased the token-based plan.

### Highlights

- **New Alibaba Cloud Bailian (Token Plan) preset**:
  - Preset ID: `bailian-token-plan`
  - Default Base URL: `https://token-plan.cn-beijing.maas.aliyuncs.com/apps/anthropic`
  - Auth scheme: `bearer` (maps to `ANTHROPIC_AUTH_TOKEN`)
  - Model mappings are left empty by default for user configuration; `default` is pre-filled with `qwen3.6-plus` as a reference.
  - Displayed alongside the existing pay-as-you-go "Alibaba Cloud Bailian" preset for easy distinction.

### Downloads

- Windows users can choose `CC-Desktop-Switch-v1.0.16-Windows-Setup.exe` or `CC-Desktop-Switch-v1.0.16-Windows-Portable.zip`.
- macOS users can choose `CC-Desktop-Switch-v1.0.16-macOS-arm64.pkg` or `CC-Desktop-Switch-v1.0.16-macOS-arm64.dmg`.
